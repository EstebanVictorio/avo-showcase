# Let's talk about it a little bit

BEM is written in the following syntactic flavor:

```css
.block__element--modifier {

}

/* Where `block` refers to an entity or first-class component... */
/* And `element` refers to a potential child or sub-component of a first-class component... */

/* Where finally, `modifier` refers to a token which -... */
/* applies a certain style to an entity (be it block or element). */
```

```css

  padding: 1rem;

  height: 200px;

  width: 100%;
  height: 100%;
}

.wrapper--ok {
  border: 1px solid green;
}

  border: 1px solid red;
}
```
