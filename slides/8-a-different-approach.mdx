# Global CSS Utilities

```css

[data-status] {
  border: 4px solid
}

[data-status=*"ok"] {
  border-color: green;
}

[data-status*="error"] {
  border-color: red;
}

[data-content*="fluid"] {
  padding: 0;
}

[data-shadow="blue"] {
  box-shadow: 1px -4px 10px 10px blue;
}

[data-text-align="right"] {
  text-align: right;
}

[data-color="white"] {
  color: whitesmoke;
}

[data-bg-color="red"] {
  background-color: tomato;
}
```
