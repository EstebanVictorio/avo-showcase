# CSS

```css
[data-image*="wrapper"] {
  padding: 1rem;
  width: 400px;
  height: 200px;
}

[data-image="img"] {
  width: 100%;
  height: 100%;
}

[data-image*="error"] {
  border: 1px solid red;
}

[data-image*="ok"] {
  border: 1px solid green;
}
```
